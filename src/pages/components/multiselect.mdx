## How to use it

### Default

The typical usage of a Multiselect component.

import MultiselectDefault from '../../examples/dropdowns/multiselect/Default.tsx';
import MultiselectDefaultCode from '!!raw-loader!../../examples/dropdowns/multiselect/Default.tsx';

<CodeExample code={MultiselectDefaultCode}>
  <MultiselectDefault />
</CodeExample>

### Overflow

A Button indicating the number of "overflowing" Tags lets users see any Tags that don't fit.

import MultiselectOverflow from '../../examples/dropdowns/multiselect/Overflow.tsx';
import MultiselectOverflowCode from '!!raw-loader!../../examples/dropdowns/multiselect/Overflow.tsx';

<CodeExample code={MultiselectOverflowCode}>
  <MultiselectOverflow />
</CodeExample>

### Size

Multiselect comes in two sizes: default or compact.

import MultiselectSizeCode from '!!raw-loader!../../examples/dropdowns/multiselect/Size.tsx';
